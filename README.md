# projet-production-hydromel
Optimisation de la fermentation alcoolique - IST Ouaga 2000 - 2025-2026
La mise en œuvre d’une technologie de production d’hydromel à base de produits locaux constituera une voie prometteuse pour la valorisation des ressources agroalimentaires nationales telles que le miel et les fruits locaux. Cette étude permettra de démontrer qu’il est techniquement possible d’élaborer des hydromels fermentés de qualité à partir de formulations optimisées, tout en tenant compte des critères physico-chimiques et sensoriels essentiels à la satisfaction des consommateurs
